## 🌟 Project Overview

**SoftClub CRM** is a comprehensive, enterprise-grade Customer Relationship Management system purpose-built for **education centers, coding bootcamps, language schools, and training academies**. It replaces fragmented spreadsheets, disconnected tools, and manual processes with a single, unified platform that manages the entire student lifecycle — from the moment a lead submits an inquiry to the day they graduate and land a job.

### The Problem

Education centers face unique operational challenges that generic CRMs cannot solve:

| Challenge | Impact |
|---|---|
| **Scattered student data** | Admins waste hours searching across spreadsheets, messengers, and paper records |
| **No enrollment visibility** | Management can't track lead-to-student conversion or enrollment trends |
| **Manual attendance tracking** | Mentors spend time on logistics instead of teaching |
| **Financial blind spots** | Tuition fees, debts, expenses, and salary calculations are error-prone |
| **No graduate tracking** | Centers lose touch with alumni and can't measure employment outcomes |
| **Communication gaps** | Important messages get lost; parents and students are out of the loop |

### The Solution

SoftClub CRM solves all of these with a centralized platform that provides:

- ✅ **Complete student lifecycle management** — from lead capture to alumni tracking
- ✅ **Real-time dashboard and analytics** — instant visibility into KPIs across all branches
- ✅ **Financial management** — tuition fees, payments, debts, expenses, budgets, and salary
- ✅ **Group & schedule management** — timetables, journals, attendance, and progress tracking
- ✅ **Role-based access** — every user sees only what they need
- ✅ **Multi-branch support** — manage multiple locations from one dashboard
- ✅ **Multi-language interface** — English, Russian, and Tajik out of the box
- ✅ **SMS campaigns** — reach students and leads with targeted messaging

## 🔐 Security

SoftClub CRM implements enterprise-grade security at every level:

### Authentication

| Feature | Description |
|---|---|
| **JWT Authentication** | Secure token-based authentication using industry-standard JSON Web Tokens |
| **Refresh Token Flow** | Automatic token renewal for seamless sessions without security compromise |
| **Auto-Logout** | Token expiration check on every page load — expired sessions are terminated |
| **Secure Token Storage** | Tokens stored in localStorage with XSS protection via HTTP interceptors |
| **Login/Logout Management** | Clean session handling with token creation and destruction |

### Role-Based Access Control (RBAC)

SoftClub CRM implements a granular 7-role permission system:

| Role | Access Level |
|---|---|
| **SuperAdmin** | Full system access — all modules, all branches, all data |
| **Admin** | Full operational access — students, groups, mentors, courses, administration |
| **Mentor** | Groups they teach, timetable, their profile, teaching materials |
| **Student** | Their groups, timetable, homework, profile, rewards |
| **Parent** | Their child's groups, mentors, and timetable |
| **Accounting** | Financial modules — payments, salary, advances |
| **Marketing** | Courses catalog and lead management |

### Dynamic Route Protection

- Routes are dynamically built based on the logged-in user's role(s)
- Users with multiple roles receive a merged set of accessible routes
- Attempting to access unauthorized routes redirects to the appropriate default page
- Sidebar navigation adapts to show only permitted sections

### Audit & Compliance

| Feature | Description |
|---|---|
| **Activity Logs** | Complete audit trail of all system actions accessible under Administration |
| **Session Management** | Admin ability to manage user accounts and access |
| **Data Validation** | Form validation with Formik + Yup for data integrity |
